# Davide Santangelo - Terminal Portfolio

![License](https://img.shields.io/github/license/davidesantangelo/davidesantangelo.com)

A unique, interactive terminal-based portfolio website. Discover information about me by typing commands, just like a real developer.

## Features

- **Retro Terminal Interface**: A fully functional command-line interface in the browser.
- **Interactive Commands**: Type `ls`, `cat`, `help` and more to navigate.
- **Persistence**: Remembers your command history and theme settings using `localStorage`.
- **Customization**: Change the terminal theme with the `theme` command.
- **Easter Eggs**: Try `matrix` for a cool surprise.
- **Responsive**: Works on desktop and mobile.

## Available Commands

| Command         | Description                                        |
| --------------- | -------------------------------------------------- |
| `help`          | List all available commands                        |
| `ls`            | List virtual files                                 |
| `cat [file]`    | Read the content of a file (e.g., `cat about.txt`) |
| `clear`         | Clear the terminal screen                          |
| `whoami`        | Display current user                               |
| `date`          | Show current date and time                         |
| `open [url]`    | Open an external URL                               |
| `socials`       | List all social media links                        |
| `theme [color]` | Change theme (green, amber, blue, white)           |
| `history`       | Show command history                               |
| `matrix`        | Enter the Matrix...                                |
| `reboot`        | Reboot the system (reload page)                    |

## Technologies

- **HTML5 & CSS3**: For the terminal structure and styling (CRT effects, glowing text).
- **JavaScript (Vanilla)**: For the command parser, file system logic, and interactivity.
- **Fira Code**: Modern monospace font for a crisp developer experience.
